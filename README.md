## 🛠️ 사용 기술 및 라이브러리

- Android Studio, SQLite
- Google Map Api , 공공 데이터

## 🙋🏻‍♂️ 담당한 기능

- Android를 활용한 화면 구성 및 기능 구현
- SQLite를 이용한 내장 DB 연동
- Google PlayStore을 이용한 어플리케이션 배포
